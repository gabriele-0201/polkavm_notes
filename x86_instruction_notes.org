** General
x86-64 instruction may be at most 15 bytes in length, made by the following structure:

| legacy prefix        | <-- lsb |
| opcode with prefixes |         |
| ModR/M               |         |
| SIB                  |         |
| Displacement         |         |
| Immediate            | <-- msb |
